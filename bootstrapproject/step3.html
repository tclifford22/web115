This is the step of adding color. To do this I use gradient mapping which is something you can find in photoshop. This technique can colorize ones black and white line art with ease, without muddying or dulling the image with only one color. It uses a gradient effect to more evenly and naturally color your designs.

The Barbaric King

At this stage the black and white value shading you worked on previously is holding all of the information and letting the program work for you. Another key element to this is that if you color your character or object and end up wanting to change it, this is SO MUCH EASIER and saves you SO MUCH TIME. If you were to shade and color it right from the start, keeping the natural feel along with changing the color would be SO much more difficult.
